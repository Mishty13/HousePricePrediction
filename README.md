# HousePricePrediction
House Price Prediction using Machine Learning techniques including Linear Regression and Random Forest Regressor. The project involves data preprocessing, exploratory data analysis, visualization, model evaluation, and feature importance analysis using Python, Pandas, Scikit-learn, Matplotlib, and Seaborn.

## Project Overview
This project predicts house prices using Machine Learning techniques. The dataset was preprocessed, analyzed, and used to train regression models for accurate price prediction.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Jupyter Notebook

## Models Used
- Linear Regression
- Random Forest Regressor

## Evaluation Metrics
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- R² Score

## Project Structure
- analysis.ipynb
- Housing.csv
- summary.pdf/docx
- charts/

## Key Findings
- Area was the most important feature affecting house prices.
- Bathrooms, parking, air conditioning, and number of stories also influenced prices.
- Linear Regression performed better than Random Forest on this dataset.
